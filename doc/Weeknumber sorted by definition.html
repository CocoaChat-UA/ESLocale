Legend
LCID: language identifier.
Code: Abbreviated name of the language. In most cases it is created by taking the two-letter language abbreviation from the ISO Standard 639 and adding a third letter, as appropriate, to indicate the sublanguage.
NT: Windows NT 4.0
98: Windows 98 SE
2k: Windows 2000 Professional
XP: Windows XP Professional Version 2002
EU: Country is member of the European Union. A EU member country should realize the EN 28601 (correspond ISO 8601).
ISO: The week number definition correspond ISO 8601.
St:
State:   unknown    correct, confirmed    presumably correct    wrong, confirmed    presumably wrong
